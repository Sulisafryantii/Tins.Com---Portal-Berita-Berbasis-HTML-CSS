# Tins.Com - Mockup Portal Berita (HTML & CSS)

## 📖 Tentang Proyek
Tins.Com adalah sebuah proyek antarmuka web (User Interface) statis yang didesain menyerupai portal berita modern. Proyek ini dibuat untuk mempraktikkan dasar-dasar penyusunan tata letak halaman web menggunakan elemen semantik HTML dan styling CSS.

## ✨ Fitur Tampilan
* **Navigasi Utama:** Bar navigasi interaktif dengan kategori berita (Home, Technology, Healthy, Business) dan simulasi kolom pencarian.
* **Sorotan Utama (Hero Section):** Menampilkan berita utama atau *headline* dengan gambar ukuran besar agar menarik perhatian pembaca.
* **Grid Artikel:** Kumpulan berita terkini yang disusun menggunakan sistem *grid/layout* yang rapi sehingga mudah dibaca.
* **Kategori Konten:** Pemisahan bagian berita berdasarkan topik spesifik (misalnya: *Healthy, Fashion*, dll).

## 🛠️ Teknologi yang Digunakan
* **HTML5:** Digunakan untuk membangun kerangka dan struktur dasar konten website.
* **CSS3:** Digunakan untuk mengatur tata letak, warna (dominasi tema biru), ukuran gambar, dan tipografi agar tampilan web terlihat profesional.

## 🚀 Cara Melihat Hasilnya
Karena proyek ini adalah file web statis sederhana, Anda tidak perlu menginstal aplikasi khusus atau menjalankan server.
1. Unduh atau salin file `index.html` (dan file CSS/gambar jika ada).
2. Klik dua kali pada file `index.html` tersebut.
3. Halaman web akan otomatis terbuka di browser bawaan komputer Anda (seperti Google Chrome, Firefox, atau Safari).
